# Учебная практика
Игра написана на движке Unity на языке программирования C# в рамках учебной практики в Санкт-Петербургском Государственном Университете Аэрокосмического Приборостроения(СПБГУАП) студентом группы 4526 1 курса направления 09.03.02(информационные системы и технологии в дизайне) Петровым Андреем.
# Суть игры
Основной геймплей заключается в том, чтобы прожить 3 минуты, паралелльно убивая монстров и получая опыт. С врагов выпадают кристалы с опытом, а с объектов окружения лечебные зелья. При повышении уровня вам на выбор дадут 3 улучшения, среди которых либо новые предметы и оружие, либо улучшение старых.
# Запуск
Чтобы запустить игру необходимо открыть файл VampSurv.exe в папке Build 5
# Управление
wasd - передвижение
мышь - навигация в меню
esc - пауза
